# Order book

## install
prerequisites: `pip install pipenv`
```
git clone https://github.com/marcinliebiediew/order_book
cd order_book
pipenv install
```
## run
```
pipenv run python src/app.py
```
or 
```
pipenv run python src/app.py ./tests/test_data/1.in
```
